Storage Optimizer Pro

Storage Optimizer Pro is a lightweight desktop utility that helps users free up disk space by detecting and removing unnecessary files from their system. The application scans the computer for temporary files, cache files, and other useless data that take up valuable storage.

This tool is designed to make storage management simple, fast, and safe for everyday users.

Features

✨ Junk File Detection
Scans the system for temporary and unnecessary files that waste storage space.

🧹 Easy Cleanup
Quickly remove useless files to free up disk space.

⚡ Fast Scanning Engine
Efficient scanning system that analyzes storage usage quickly.

📊 Storage Optimization
Helps users keep their system clean and optimized.

🖥️ Simple Interface
Easy-to-use design suitable for both beginners and advanced users.

How to Run the Project

Make sure Python 3 is installed on your system.

1. Clone the repository
git clone https://github.com/BHAvyA-JAin-Dev/Storage-Optimizer-Pro.git
2. Open the project folder
cd Storage-Optimizer-Pro
3. Run the application
python main.py
Build the Executable (.exe)

To convert the application into a Windows executable file:

Run the build script:

build.bat

After the build process finishes, the .exe file will be generated.
